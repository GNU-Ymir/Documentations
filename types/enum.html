Enum

Enumeration

Enumerations are user-defined types that have a set of values. The keyword enum is used to define an enumerated type, from which the field type is deduced at compile time. It is mandatory that each field shares the same type.

import std::io

enum
| MONDAY = "Mon"
| TUESDAY = "Tue"
| WEDNESDAY = "Wed"
| THURSDAY = "Thu"
| FRIDAY = "Fri"
| SATURDAY = "Sat"
| SUNDAY = "Sun"
 -> Day;

def foo (day : Day) {
    println (day);
}

def main () {
    let d = Day::MONDAY;
    inspect (d);
}

Enum can be cast to the type of its value, but the reverse is not possible.

import std::io

enum
| MONDAY = "Mon"
| TUESDAY = "Tue"
| WEDNESDAY = "Wed"
| THURSDAY = "Thu"
| FRIDAY = "Fri"
| SATURDAY = "Sat"
| SUNDAY = "Sun"
 -> Day;

def foo (day : [c32]) {
    println (day);
}

def bar (day : Day) {
    println (day);
}

def main () {
    foo (Day::MONDAY);
    bar (Day::MONDAY);
    // Try to add the line 
    // bar ("Mon")
}
