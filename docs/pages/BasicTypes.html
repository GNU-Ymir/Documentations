2. Basic types

Ymir provides a number of types which have the same size regardless of the platform.
  • Signed Integers byte, short, int, long
  • Unsigned Integers ubyte, ushort, uint, ulong
  • Floating point float
  • UTF-8 characters char
  • bool either true or false
All this types can provide an initialization property (among others) ::init, as shown in the following example.

Variable types are always infered. Numbers can be optionally annotated via a suffix.

Some types also provide other properties

Here are the decimals properties :
  • ::max : the maximum value of an integer number
  • ::min : the minimum value of an integer number
  • ::sizeof : the memory size of an integer (in bytes)
  • ::typeid : the string representration of the int type
Here are the floats properties :
  • ::init : 0.0f
  • ::max : the maximum value of a floating number
  • ::min : the minimum value of a floating number
  • ::sizeof : the memory size of an float (in bytes)
  • ::dig : the number en precision decimal digits
  • ::epsilon : the smallest possible increment to the value 1
  • ::mant_dig : the number of bits in the mantis
  • ::max_10_exp : the largest value such that 10^max_10_exp is representative
  • ::max_exp : the max value such that 2^max_exp is representative
  • ::min_10_exp : the minimum value such that 10^min_10_exp is representative
  • ::min_exp : the minimum value such that 2^min_exp is representative
  • ::infinity : the value that represents infinity in floating number
  • ::typeid : the string representration of the float type
